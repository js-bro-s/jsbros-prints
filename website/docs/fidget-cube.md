# Fidget Cube

A six-sided desk fidget with a different action on each face — click, spin, roll, flip, slide, and rub.

![Fidget Cube](pathname:///img/prints/fidget-cube.png)

## Details

| Field | Value |
|-------|-------|
| **Price** | $2.00 |
| **Print Time** | ~3 hours |
| **Material** | PLA |
| **Dimensions** | 30mm × 30mm × 30mm |
| **Infill** | 15% |
| **Supports** | No |

## Description

Each face has a unique tactile feature. The click side has two buttons with a satisfying snap, the spin side has a small disc that rotates freely, and the roll side has three metal-ball-sized grooves. Great for keeping your hands busy during class (quietly).

## Print Notes

- Sand the buttons smooth after printing so they don't catch
- Test all 6 sides click/spin before selling
- Print upright with the click-side facing up for best button quality
