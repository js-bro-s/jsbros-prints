# Phone Stand

A compact desk stand that holds your phone at the perfect angle for watching videos or video calling.

![Phone Stand](pathname:///img/prints/phone-stand.png)

## Details

| Field | Value |
|-------|-------|
| **Price** | $2.50 |
| **Print Time** | ~1.5 hours |
| **Material** | PLA |
| **Dimensions** | 70mm × 60mm × 55mm |
| **Infill** | 20% |
| **Supports** | No |

## Description

Holds any phone in portrait or landscape mode. The front lip keeps the phone from sliding, and the back angle is set at 65° for comfortable viewing on a desk. Flat bottom with rubber-pad grooves so it doesn't slide around. Works with most cases on.

## Print Notes

- Print upright for strongest structure
- Add small adhesive rubber pads in the base grooves for grip
- Test with at least 3 different phone sizes before selling
