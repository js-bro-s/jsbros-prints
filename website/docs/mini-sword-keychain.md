# Mini Sword Keychain

A tiny detailed sword with a loop at the hilt for attaching to a keychain, backpack zipper, or lanyard.

![Mini Sword Keychain](pathname:///img/prints/mini-sword-keychain.png)

## Details

| Field | Value |
|-------|-------|
| **Price** | $2.00 |
| **Print Time** | ~1 hour |
| **Material** | PLA |
| **Dimensions** | 70mm × 15mm × 8mm |
| **Infill** | 20% |
| **Supports** | No |

## Description

Designed under 80mm so it fits comfortably on a keychain ring. The hilt has a 3mm loop hole for a standard split ring. Blade has engraved line details on both sides. Looks great in metallic silver or gold PLA.

## Print Notes

- Keep total length under 80mm for keychain comfort
- Print flat on the bed for best blade detail
- Use 0.12mm layer height for sharper engraving lines
- Thread a split ring through the hilt loop before selling
