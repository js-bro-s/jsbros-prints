# Spinning Top

A classic twist-and-spin top that keeps going. Simple, cheap, and surprisingly fun to battle with friends.

![Spinning Top](pathname:///img/prints/spinning-top.png)

## Details

| Field | Value |
|-------|-------|
| **Price** | $1.00 |
| **Print Time** | ~45 min |
| **Material** | PLA |
| **Dimensions** | 35mm diameter × 40mm tall |
| **Infill** | 25% |
| **Supports** | No |

## Description

Weighted at the rim for maximum spin time. The pointed tip is printed with a slight flat so it doesn't scratch desks. Challenge your friends — longest spin wins.

## Print Notes

- Higher infill (25%) makes it spin longer and more stable
- Test on a flat surface before selling
- Print with the tip pointing up for a clean point
