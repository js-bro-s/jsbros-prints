# Click Button Fidget

A single oversized button that clicks with a deep, satisfying snap. The simplest fidget we make — and one of the most popular.

![Click Button Fidget](pathname:///img/prints/click-button-fidget.png)

## Details

| Field | Value |
|-------|-------|
| **Price** | $1.50 |
| **Print Time** | ~30 min |
| **Material** | PLA |
| **Dimensions** | 40mm diameter × 15mm tall |
| **Infill** | 15% |
| **Supports** | No |

## Description

One button, one purpose: click it. The dome-shaped cap sits on a printed spring mechanism that gives a tactile pop every press. Small enough to hide in your palm during class. The click is quiet enough to not get you in trouble (mostly).

## Print Notes

- Print the cap and base as one piece — the spring is built into the geometry
- 0.2mm layer height works fine
- Test the click 50+ times before selling to make sure it doesn't flatten out
