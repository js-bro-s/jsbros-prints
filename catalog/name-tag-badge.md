# Name Tag Badge

A personalized name badge with a clip slot on the back. Custom-printed with the buyer's name — our first made-to-order item.

## Details

| Field | Value |
|-------|-------|
| **Price** | $2.00 |
| **Print Time** | ~45 min |
| **Material** | PLA |
| **Dimensions** | 70mm × 25mm × 5mm |
| **Infill** | 15% |
| **Supports** | No |

## Description

Each badge is printed with raised lettering of the buyer's name. The back has a horizontal slot that fits a standard badge clip or safety pin. Offered in any PLA color we have in stock. Great for backpacks, lanyards, or locker tags.

## Print Notes

- Customize the STL in Tinkercad before each print — swap out the name text
- Print text-side up for crisp letters
- 0.12mm layer height recommended for name legibility
- Keep a list of orders so you don't mix them up
