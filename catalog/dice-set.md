# Dice Set

A matching set of D6 and D20 dice for tabletop games, board games, or just deciding who goes first. Printed with recessed numbers for easy reading.

## Details

| Field | Value |
|-------|-------|
| **Price** | $3.00 (set of 2) |
| **Print Time** | ~2 hours (both) |
| **Material** | PLA |
| **Dimensions** | D6: 16mm / D20: 20mm |
| **Infill** | 100% |
| **Supports** | Yes (D20) |

## Description

The D6 is a standard six-sided die with recessed pips. The D20 has recessed numbers 1–20 for RPG and tabletop games. Both are printed solid (100% infill) so they roll with good weight and feel fair. Fill the recessed numbers with a contrasting paint pen for a clean look.

## Print Notes

- 100% infill is critical — hollow dice don't roll evenly
- D20 needs supports; orient with a flat face down
- Use a paint pen or acrylic to fill recessed numbers
- Sand lightly for even rolling
