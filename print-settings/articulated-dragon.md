## Slicer Settings

| Setting | Value |
|---------|-------|
| Layer height | 0.2mm |
| Infill | 15% |
| Infill pattern | Grid |
| Supports | **NO** (print-in-place — supports will fuse joints) |
| Brim | Yes (recommended for bed adhesion) |
| Print speed | 40mm/s |
| Nozzle temp | 210°C |
| Bed temp | 60°C |
| Material | PLA (silk PLA recommended) |

**Critical:** Do NOT enable supports. This is a print-in-place design.
