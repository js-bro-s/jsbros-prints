# Designs

This folder is for design notes, sketches, and Tinkercad links — before the STL is ready.

## How to Add a Design Idea

1. Create a new file: `designs/item-name.md`
2. Fill in the template below
3. Commit: `add design idea: item-name`

---

## Design Template

```md
# Design: Item Name

**Idea from:** [your name]
**Date:** YYYY-MM-DD
**Status:** Idea / Sketching / In Tinkercad / Ready to print

## Description
What is it? What does it do?

## Reference Images
- Link or description of what you're basing it on

## Tinkercad Link
*(paste link once started)*

## Dimensions (target)
- Width:
- Height:
- Depth:

## Notes
Anything else to remember when designing.
```
